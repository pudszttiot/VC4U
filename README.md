## Description

VC4U is a user-friendly graphical user interface (GUI) application developed in Python using PyQt5. Its primary function is to facilitate the cutting of video files with ease. Users can select input and output video files, specify the start and end times for the desired segment, and then execute the cutting process. VC4U supports popular video formats such as MP4, AVI, and MKV. Throughout the cutting process, users receive real-time feedback on the progress, and upon completion, the original and cut file sizes are displayed for reference. With its simple interface and straightforward functionality, VC4U offers an efficient solution for video editing tasks.


## Features

- Select input and output video files.
- Specify start and end times for cutting.
- Supports MP4, AVI, and MKV video formats.
- Provides feedback on the cutting process.
- Displays original and cut file sizes.
